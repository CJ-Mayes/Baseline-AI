Tableau Integration

Following the steps from the Information Lab, I was able to add the Wimbledon chatbot to this webpage. I've disabled the chatbot for now, but you can still see the code in the repo.

Response Ready

With Tableau Langchain, you can ask questions about your data and get answers in real time. The chatbot connects directly to the data stored on the server to produce a response. It will only query the dataset passed to it.

Design & Host

I've mirrored Wimbledon's website with the color scheme, and green banner bar at the top - all whilst adding my own unique touch.
The site is hosted through Render, directly from the GitHub repository.
